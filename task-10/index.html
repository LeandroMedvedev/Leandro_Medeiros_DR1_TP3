Quiz | Media Queries

Qual a principal função das Media Queries?

  1. Alterar o layout da página de acordo com o dispositivo.
  2. Alterar o ícone do cursor quando o usuário passa o mouse sobre um elemento.
  3. Criar animações complexas.
  4. Gerenciar o carregamento de fontes.
  5. Definir a paleta de cores da página.

A resposta correta é a alternativa A: Alterar o layout da página de acordo com o dispositivo. As Media Queries permitem que você crie diferentes estilos para diferentes tamanhos de tela, garantindo uma experiência de usuário otimizada em diversos dispositivos.
