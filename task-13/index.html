1. Qual a principal vantagem de adotar a metodologia mobile-first no desenvolvimento de um site?

  1. Prioriza a experiência do usuário em dispositivos de tela grande.
  2. Diminui o tempo de desenvolvimento para dispositivos móveis.
  3. Garante que o site seja acessível apenas em dispositivos móveis.
  4. Facilita a adaptação do design para diferentes tamanhos de tela.

Resposta: A resposta correta é a alternativa D: Facilita a adaptação do design para diferentes tamanhos de tela. Ao começar o desenvolvimento pelo mobile, o designer se concentra nas funcionalidades e elementos mais essenciais, o que facilita a adição de elementos visuais e funcionalidades para telas maiores de forma gradual.

2. Qual das seguintes unidades de medida é mais recomendada para criar layouts responsivos?

  1. Pixels (px)
  2. Pontos (pt)
  3. Porcentagem (%)
  4. Centímetros (cm)

Resposta: A resposta correta é a alternativa C: Porcentagem (%). As porcentagens são relativas ao tamanho do elemento pai, o que permite que o layout se adapte dinamicamente a diferentes tamanhos de tela.

3. Qual a função principal das media queries?

  1. Criar animações CSS.
  2. Definir estilos específicos para diferentes dispositivos e tamanhos de tela.
  3. Gerenciar o carregamento de fontes.
  4. Controlar a responsividade de imagens.

Resposta: A resposta correta é a alternativa B: Definir estilos específicos para diferentes dispositivos e tamanhos de tela. As media queries permitem que você aplique estilos CSS diferentes com base em características do dispositivo, como largura da tela, orientação e resolução.

4. Qual é a importância do viewport meta tag em um site responsivo?

  1. Define o zoom inicial da página.
  2. Especifica a largura da tela do dispositivo.
  3. Indica o tipo de dispositivo que está acessando a página.
  4. Controla o carregamento de scripts.

Resposta: A resposta correta é a alternativa A: Define o zoom inicial da página. A viewport meta tag controla a largura e a escala inicial da página, garantindo que ela se ajuste corretamente à tela do dispositivo.

5. Qual das seguintes afirmações sobre o mobile-first é FALSA?

  1. O mobile-first prioriza a experiência do usuário em dispositivos móveis.
  2. O mobile-first simplifica o desenvolvimento de sites responsivos.
  3. O mobile-first ignora completamente o design para desktops.
  4. O mobile-first é uma abordagem recomendada para a maioria dos projetos web.

Resposta: A resposta correta é a alternativa C: O mobile-first ignora completamente o design para desktops. O mobile-first não ignora o design para desktops, mas sim o utiliza como uma extensão do design mobile, adicionando elementos e funcionalidades conforme necessário para telas maiores.
